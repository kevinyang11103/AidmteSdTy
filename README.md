# 项目简介

基于深度学习的棉花品种分类检测系统，采用先进的YOLOv10算法，结合自制的YOLO数据集进行模型训练。该系统包含功能模块如实时监测、图像识别、品种分类等，通过UI界面实现用户友好交互。技术特点包括高检测速度、准确率高、易于部署，且提供完整的Python项目源码及训练好的模型，助力科研与农业自动化领域对棉花品种的快速分类与检测。


## 核心功能
- 实时监测棉花品种
- 图识别与分析
